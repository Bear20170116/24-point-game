# 24-point-game
经典的24点数学游戏，使用HTML、CSS和JavaScript开发
## 功能特点
- 随机生成4个数字
- 实时验证答案
- 提供解题提示
- 游戏统计和历史记录
- 移动端优化，包含虚拟键盘
- PWA支持，可安装到主屏幕
- 完全离线可用

## 如何使用
1 直接打开`index.html`文件
2 或使用本地服务器运行：`python -m http.server 8000`
3 在浏览器中访问：`http://localhost:8000`

## 技术栈
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- PWA (Progressive Web App)
